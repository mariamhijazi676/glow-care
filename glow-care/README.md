# Glow & Care

## Project Description
Glow & Care is a responsive beauty and self-care website built using ReactJS.  
The website helps users explore beauty tips, skin care routines, hair care ideas, and self-care habits.

## Features
- Home page
- About page
- Services page
- Contact page
- Responsive design for desktop and mobile devices

## Technologies Used
- ReactJS
- CSS
- Vite
- Git & GitHub

## Setup Instructions
1. Download the project
2. Open the project in VS Code
3. Run:
npm install
4. Run:
npm run dev
